# Find My Weather

![Weather App image](https://github.com/niloyKGhosh/weather-app/blob/master/Weather-App.png)

###### This is a weather app developed using the [Weather Stack API](https://weatherstack.com/).

## Getting Started

1. Install dependencies
    ```bash
    npm install
    ```
2. Start the server

    ```bash
    npm run dev
    ```
